Clock

this code displays the time in different time zones. to use, copy the code below and add into your site.

to set a timezone, add a P element and give it the class "clock", or "clockseconds". add the attribute "timezone" and set that to whatever timezone suits your needs. here is a list of timezones you can reference.

eg: <p class="clock" timezone="+12"><p>

eg: <p class="clockseconds" timezone="-4"><p>

if you have the class "clockseconds", it displays the seconds.
